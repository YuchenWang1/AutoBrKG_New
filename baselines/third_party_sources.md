# Third-Party Baseline Sources

Third-party framework source code is not vendored in this open-source package. Use the official repositories below and apply the prompts/settings in this directory.

| Baseline | Source | What is included here |
| --- | --- | --- |
| LightRAG | https://github.com/HKUDS/LightRAG | Experiment note and output-format alignment instructions |
| KGGEn / KGGen | https://github.com/stair-lab/kg-gen | Experiment note and output-format alignment instructions |
| EDC | https://github.com/clear-nus/edc | Prompt-template settings and bridge schema pointers |

Direct LLM baselines use `prompts/llm_bridge_kg_extraction_prompt.md` with the model IDs listed in `model_matrix.json`.
